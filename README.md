SEC-LEO Document Tracking System

Project Title: SEC-LEO Document Tracking System

Purpose: This system is created for the Securities and Exchange Commission - Legazpi Extension Office to help manage and track office document records.

Module Scope: This module allows the admin to create, view, update, and delete document records. It also supports file uploads, document status updates, and dashboard summary counts.

Stack

1. Frontend: HTML, CSS, JavaScript

2. Backend: PHP

3. Database: MySQL

4. Local Server: XAMPP


Features

1. Admin login and registration

2. Add document records

3. Upload document files

4. View document details

5. Update document status

6. Delete document records

7. Dashboard document counts


To run locally:

1. Clone the project.

2. Move the project folder to:  C:\xampp\htdocs\WebsysFinalProj

3. Open XAMPP.

4. Start Apache and MySQL.

5. Open phpMyAdmin:   http://localhost/phpmyadmin

6. Create a database named:    sec_leo_db

7. Import the SQL file from:  database/sec_leo_db.sql

8. Open the project in the browser:    http://localhost/WebsysFinalProj/


CRUD Functions

1. Create: Admin can add new document records.

2. Read: Admin can view document records and document details.

3. Update: Admin can update document status.

4. Delete: Admin can delete document records.
